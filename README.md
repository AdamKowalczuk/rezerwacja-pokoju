# Aplikacja do rezerwacji pokoju w hotelu

Aplikacja została wykonana przy użyciu technologii React, Redux i TypeScript. Prowizoryczna baza danych jest zapisywana w Local Storage. Aplikacji składa się tylko z części frontend

## Aplikacja składa się z 3 podstron:

### Pierwsza strona to formularz rejestracyjny. Formularz wykorzystuje wyrażenia regularne.

![mobile](https://user-images.githubusercontent.com/57939211/160442956-6149e15a-9bdc-477d-b06e-a3432fdfe00e.png)


### Druga strona przedstawia wszystkie dostępne pokoje wraz z ich opisem. Użytkownik ma możliwość filtrowania wyników według lokalizacji, ceny, liczby osób oraz liczby gwiazdek. Użytkownik może również dodaj własny pokój do bazy.
![main-desktop](https://user-images.githubusercontent.com/57939211/160442970-3cc04473-4512-4f16-9a37-ecfb926efe45.PNG)
![mobile (2)](https://user-images.githubusercontent.com/57939211/160442929-eb75a817-cadb-45b8-9825-e3609fb58199.png)
![mobile (1)](https://user-images.githubusercontent.com/57939211/160442938-06bc812e-a5a1-4419-b577-c7f2b00c876b.png)


### Trzecia strona przedstawia informacje o wybranym pokoju i umożliwia rezerwację pokoju.
